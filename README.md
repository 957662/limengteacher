# 人教版九年级物理课程实验模拟教学网站

## 项目简介
本网站用于人教版九年级物理课程的实验模拟教学，包含了从第十三章到第二十二章的所有内容。网站采用现代化的响应式设计，提供清晰的知识点讲解、物理原理分析和实验模拟入口。

## 启动方法

### 方法一：使用启动脚本（推荐）
1. 双击运行 `start_server.ps1` 脚本
2. 脚本会自动尝试使用以下方式启动服务器：
   - Python 的 http.server 模块
   - Python3 的 http.server 模块
   - Node.js 的 http-server（如果已安装）
3. 启动成功后，在浏览器中访问：http://localhost:8080

### 方法二：手动启动Python服务器
1. 确保已安装Python
2. 打开命令提示符或PowerShell
3. 导航到项目根目录
4. 运行命令：`python -m http.server 8080`
5. 在浏览器中访问：http://localhost:8080

### 方法三：使用Node.js的http-server
1. 确保已安装Node.js
2. 安装http-server：`npm install -g http-server`
3. 导航到项目根目录
4. 运行命令：`http-server -p 8080`
5. 在浏览器中访问：http://localhost:8080

### 方法四：直接在浏览器中打开
1. 找到项目根目录下的 `index.html` 文件
2. 右键点击，选择"打开方式"，使用任意浏览器打开
3. 注意：直接打开可能会导致部分功能异常，建议使用HTTP服务器方式

## 使用说明

1. **主页导航**：在主页上可以看到所有章节的卡片，点击任意章节卡片进入该章节
2. **章节内容**：每个章节包含三个部分：
   - **知识点讲解**：详细的知识点列表
   - **物理原理**：深入解释物理概念、规律和公式
   - **实验模拟**：预留了超链接位置，可替换为阿里闪灵制作的实验模拟链接
3. **侧边菜单**：在章节页面右侧（或移动端顶部）有侧边菜单，可快速跳转到章节的不同部分
4. **滚动到顶部**：页面右下角有一个向上箭头按钮，点击可快速返回页面顶部
5. **返回章节列表**：每个章节页面顶部都有"返回章节列表"按钮，点击可返回主页

## 交互功能

- **平滑滚动**：点击导航链接时平滑滚动到对应位置
- **滚动监听**：自动高亮当前阅读的章节部分
- **响应式设计**：适配不同屏幕尺寸，在手机、平板和电脑上都有良好的显示效果
- **悬停效果**：章节卡片和按钮有悬停动画效果

## 自定义实验模拟链接

在每个章节的"实验模拟"部分，都有预留的超链接位置。您可以按照以下步骤替换为阿里闪灵制作的实验模拟链接：

1. 打开对应章节的HTML文件（例如：chapters/ch13.html）
2. 找到"实验模拟"部分（id="simulations"）
3. 将 `<a href="#" class="simulation-link...">` 中的 `href="#"` 替换为您的阿里闪灵实验链接
4. 保存文件，刷新浏览器即可看到效果

1. 确保您的浏览器支持HTML5和CSS3
2. 建议使用Chrome、Firefox、Edge等现代浏览器
3. 如果遇到页面显示异常，请尝试清除浏览器缓存
4. 直接打开HTML文件可能会导致部分功能异常，建议使用HTTP服务器方式
5. 本网站仅用于教学目的，请勿用于商业用途
