| `activeChapter` | string | undefined | 当前激活的章节ID，用于高亮侧边栏中的当前章节 | 侧边栏的章节列表中，当前章节会以蓝色高亮显示 | 在各章节页面的ChapterLayout组件调用中修改为对应章节ID（如"ch21"） | 必须与章节ID一致，否则高亮显示不正确 |
| `sectionIds` | string[] | ['knowledge', 'principles', 'simulations'] | 需要监听滚动位置的章节部分ID列表 | 侧边栏的章节导航中，当前滚动到的部分会以蓝色高亮显示 | 在ChapterLayout组件内部修改 | 必须与章节页面中的section ID一致 |
| `activeSection` | string | '' | 当前滚动到的章节部分ID，用于高亮侧边栏中的当前部分 | 侧边栏的章节导航中，当前部分会以蓝色高亮显示 | 由useScrollSpy钩子自动更新，无需手动修改 | 自动根据滚动位置更新 |

## 7. 注意事项

1. **修改前备份**：在修改任何文件之前，建议先备份原始文件，以防止修改错误导致的问题。
2. **保持代码风格一致**：修改代码时，建议保持与原有代码风格一致，包括缩进、命名规范等。
3. **测试修改效果**：修改完成后，建议在浏览器中测试修改效果，确保页面显示正常，功能正常。
4. **响应式测试**：修改完成后，建议在不同屏幕尺寸下测试页面显示效果，确保响应式设计正常。
5. **构建测试**：修改完成后，建议运行`npm run build`命令，确保项目能够成功构建，没有编译错误。
